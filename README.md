# Music-Bot
### http://t.me/neuromusicbot

Бот создан для проекта студентов третьего курса Университета ИТМО по предмету "Нейротехнологии и аффективные вычисления".

### Авторы: Серебренникова Валерия и Пряничников Кирилл.

Описание: бот позволяет пользователю добавить любимые музыкальные жанры и на их основе получить для себя музыкальный плейлист, сгенерированный нейросетью, подходящей эмоции, найденной на фотографии пользователя.
